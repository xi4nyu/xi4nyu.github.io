# Go 变量声明

## 概述
Go 是静态类型语言，变量在使用前必须声明。未赋值的变量具有默认值（零值）。变量分为包级变量和局部变量。

## 零值表
| 类型               | 默认值 |
|--------------------|--------|
| 布尔类型           | false  |
| 所有整数类型       | 0      |
| 浮点数类型         | 0.0    |
| 字符串类型         | ""     |
| 指针、函数、接口、切片、通道、映射 | nil    |

## 变量声明
### 标准声明
```go
var a int = 128
var b int8 = 6
var s string = "hello"
```

### 批量声明
```go
var a, b, c int = 1, 2, 3
var a, b, c = 1, 2, 3  // 省略类型声明
```

### 短变量声明
```go
a, b, c := 12, 'A', "hello"
```

## 变量分类

### 包级变量
- 包级可见，导出的包级变量（大写字母开头）可视为全局变量。
- 只能使用 `var` 声明，不能使用短变量声明。

#### 初声明方式
```go
// 显式类型
var a int = 128
var b int8 = 6
var s string = "hello"

// 推荐方式
var a = 13
var b = int32(17)
var f = float32(3.14)
```

#### 延迟初始化
```go
var a int
var f float64
```

### 局部变量
- 仅在函数或方法体内可见。

#### 延迟初始化
```go
var a int
var f float64
```

#### 立即初始化
```go
a, b, c := 12, 'A', "hello"
```

## 最佳实践
- 对于延迟初始化的变量，确保零值可用。
- 声明且立即初始化的局部变量优先使用短变量声明。
